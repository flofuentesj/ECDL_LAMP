# Optically-feedback-stabilized-ECDL-
A cost-effective diode laser system for high-resolution atomic spectroscopy, developed at Universidad de Concepción's LAMP laboratory using commercial components. It features an external cavity with a piezoelectric transducer for precise wavelength tuning and stabilization that achieves sub-MHz line width essential for rubidium transitions.
